# Customer Shopping Behavior Analysis

## 📊 Project Overview
This project analyzes customer shopping behavior using transactional data to uncover insights into spending patterns, customer segments, and product preferences.

The goal is to support data-driven decision-making for marketing, product strategy, and customer engagement.

---

## 🛠️ Tools & Technologies
- Python (Pandas, Data Cleaning)
- SQL (PostgreSQL)
- Power BI (Dashboard & Visualization)

---

## 📁 Dataset
- 3,900 rows
- 18 columns
- Includes:
  - Customer demographics
  - Purchase behavior
  - Product categories
  - Discounts & reviews

---

## 🔍 Key Business Questions
- Do subscribed customers spend more?
- Which products perform best?
- How do discounts affect purchasing behavior?
- What customer segments generate the most revenue?

---

## 📈 Key Insights
- Loyal customers represent the majority of revenue
- Discounts drive purchases but not always higher spending
- Certain product categories dominate sales performance
- Age groups show different purchasing behaviors
